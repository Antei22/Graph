# Реализация графа на Java

Этот проект представляет собой реализацию структуры данных **Граф** (ориентированного и неориентированного) на Java без использования готовых коллекций. Проект включает основные методы для работы с графом, такие как добавление, удаление вершин и рёбер, а также обходы графа.

## Функциональность

### Методы и их описание:

- addVertex(int number): Добавляет вершину с заданным номером в граф.
- addEdge(int from, int to, int weight): Добавляет ребро между вершинами from и to с указанным весом.
- removeVertex(int number): Удаляет вершину с заданным номером из графа.
- removeEdge(int from, int to): Удаляет ребро между вершинами from и to.
- List<Integer> getAdjacent(int number): Возвращает список смежных вершин для заданной вершины.
- int[] dfs(int start): Выполняет обход графа в глубину (DFS), начиная с указанной вершины.
- int[] bfs(int start): Выполняет обход графа в ширину (BFS), начиная с указанной вершины.

## Как использовать

1. Склонируйте репозиторий на ваш компьютер.
2. Запустите класс `Main` и следуйте инструкциям в консоли.

## Требования

- JDK 11 или выше.
